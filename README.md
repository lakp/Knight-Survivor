# lakp.github.io
d / right = go right
a / left = go left
w / up = jump
s / down = crouch
j / z = attack
k / x = roll
p = pause
m = toggle map editor
z = zoom in/out map editor

attack projectiles to deflect them
crouch + roll = drop down walkway
